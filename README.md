## The Brain вҖ” LiquidBrain

8 cognitive modes that the agent auto-selects based on the task:

| Mode | Temp | Focus | Best For |
|------|------|-------|----------|
| **SOVEREIGN** | 0.1 | Command | Leadership, final decisions |
| **WARRIOR** | 0.3 | Execution | Hitting trades, taking action |
| **GHOST** | 0.2 | Stealth | Dark Forest, zero-footprint ops |
| **ORACLE** | 0.7 | Prediction | Research, market analysis |
| **CREATOR** | 0.9 | Generation | Building, coding, creating |
| **ARCHITECT** | 0.4 | Structure | Planning, system design |
| **SAGE** | 0.5 | Reflection | Review, learning from mistakes |
| **PHANTOM** | 0.8 | Simulation | Testing scenarios, war games |

6 reasoning frameworks: **OODA**, **PDCA**, **MARS**, **OUROBOROS**, **CYCLIC**, **SELF_REF**

## Multi-LLM Fallback

```
DeepSeek вҶ’ (fails?) вҶ’ Groq вҶ’ (fails?) вҶ’ Gemini вҶ’ (all down?) вҶ’ Graceful message
```

Random rotation prevents provider dependency. One goes down, the next picks up.

## Features

- **Self-Comitting** вҖ” pushes its own progress to GitHub
- **Dark Forest Mode** вҖ” silent, invisible, leaves no trace
- **Checkpoint System** вҖ” survives restarts via Apollo-X patterns
- **Task Queue** вҖ” priority-based, self-retrying
- **Skill Claw** вҖ” self-expanding skill library (OpenClaw compatible)
- **Evolution** вҖ” the agent improves itself over time
